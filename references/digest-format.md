# 日报格式

Markdown 日报由 `new_only.json`（`scripts/dedupe.py` 的输出）渲染。渲染工作由 `scripts/digest.py render` 完成，SKILL.md 不需要手写格式化逻辑。标题里的季节标签、岗位方向标签都是 `digest.py` 从 `config.json` 的 `target_season_label`/`job_category_label` 读出来拼的，不是写死的字符串——下面模板里的"2026秋招"、"财务"只是默认配置下的示例值。

## Markdown 日报模板（有新增时）

```
# 2026秋招 财务方向岗位监控日报 · 2026-07-08

发现 3 家新开放企业，共 5 个新增岗位

## 某某银行
- **财务共享中心-核算岗（管培生）** — 上海
  亮点：CPA优先，2027届
  链接：https://...
  来源：牛客网 · 发现于 2026-07-08

## 某咨询公司
- **审计助理（校招）** — 全国多地
  亮点：四大之一，接受应届生
  链接：https://...
  来源：应届生求职网 · 发现于 2026-07-08
```

## Markdown 日报模板（无新增时）

```
# 2026秋招 财务方向岗位监控日报 · 2026-07-08

今日未发现新的财务方向正式校招岗位。
当前已监控 12 家公司 / 27 个历史岗位。
```

这个"无新增"分支必须始终输出，不能空手退出。
